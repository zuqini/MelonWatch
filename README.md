# MelonWatch
An OpenGL FPS Platformer created for my CS488 Final Project

The following objectives were implemented:
- Texture Mapping
- Particle Systems
- Shadow mapping
- Static/Dynamic Collision
- Physics Engine
- 3D Sound
## Building
Currently only supported on OS X:

```
$ cd MelonWatch/
$ premake4 gmake && make
$ cd src/
$ premake4 gmake && make
$ ./A3
```
